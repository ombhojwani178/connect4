# AI-Based Connect 4 🔴🟡

## Brief
This is a Python-based implementation of the classic Connect 4 game, featuring an AI opponent. 

The primary aim of this project is to demonstrate how the **Minimax algorithm**, combined with **Alpha-Beta Pruning**, can be implemented to create a smart, decision-making game agent using Game Trees.

## Technologies Used
* **Python 3:** Core programming language.
* **NumPy:** Used for efficient matrix operations and implementing the core game board logic.
* **Pygame:** Used to render the graphical user interface (GUI) and detect human mouse clicks.

## Project Structure
The entire game logic and UI are contained within a single file:
* `connect4.py`

## How to Run
To execute the game, ensure you have Python and the required libraries installed, then run the following command in your terminal:

```bash
python connect4.py
